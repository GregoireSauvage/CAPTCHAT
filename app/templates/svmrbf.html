Version 1

Ce modèle a été entrainé uniquement avec des features/indicateurs extraits des la position de la souris (x, y, temps) lors du mouvement pour cliquer sur le bouton.

Features utilisées (16):

  • vitesse_moyenne
  • vitesse_min
  • vitesse_max
  • vitesse_ecart_type
  • vitesse_coefficient_variation
  • vitesse_range_ratio
  • acceleration_moyenne
  • acceleration_min
  • acceleration_max
  • acceleration_ecart_type
  • distance_totale
  • distance_ligne_droite
  • ratio_linearite
  • nombre_changements_direction
  • variance_angles
  • entropie_angles

Le dataset d'entrainement contient 550 exemples positifs et 550 exemples négatifs, dont 80% pour l'entrainement et 20% pour le test.
Les données "robot" ont été générées via plusieurs scripts python qui simulent un mouvement de souris (mouvement linéaire, aléatoire, zigzag, bézier, spline cubique, etc.).
